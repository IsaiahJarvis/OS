Use console to make either file an executable.

Program is a Floyd-Warshall all-pairs-shortest-path algorithm using threads and semaphores.

input Graph structure: The first line contains 2 integers N and M. N is the number of nodes. M is the number of undirected edges. 
Then each entry contains the link information (M entries). Line i+1 contains 3 integers ui, vi and wi which represents an undirected edge 
between nodes ui, vi and wi is the weight of that edge. Use positive edge weights.

see .txt file for example input.
